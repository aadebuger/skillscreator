🔒 隐私政策

最后更新：2025年2月

✅ 核心承诺

Skill Creator 不收集、存储或传输任何用户个人数据到我们的服务器。

1. 概述

Skill Creator（以下简称"本应用"）是一款帮助用户通过 AI 对话创建 Agent Skills 的 iOS 应用。我们非常重视您的隐私，本政策说明我们如何处理您的信息。

2. 信息收集

本应用不收集任何个人信息。

具体说明：

  • 不收集姓名、邮箱、电话等个人身份信息
  • 不收集设备标识符或广告 ID
  • 不使用任何分析或追踪 SDK
  • 不包含第三方广告

3. 本地存储

以下数据仅存储在您的设备本地：

  • API 配置 - 您配置的 LLM 服务商 API Key 和设置
  • 对话历史 - 您与 AI 的对话记录
  • 工作区文件 - AI 生成的 Skill 文件
  • 会话记录 - 创建 Skill 的历史会话

这些数据存储在 iOS 应用沙盒中，其他应用无法访问。卸载应用将永久删除所有本地数据。

4. 网络通信

本应用会进行以下网络请求：

  • LLM API 调用 - 将您的对话发送到您配置的 AI 服务商（如 DeepSeek、Kimi、智谱、OpenAI 等）

重要说明：

  • 网络请求仅发送到您自行配置的第三方 AI 服务
  • 我们不运营任何后端服务器
  • 您的 API Key 和对话内容不会经过我们的服务器
  • 请参阅您所使用的 AI 服务商的隐私政策

5. 第三方服务

本应用支持以下第三方 LLM 服务（需用户自行配置）：

  • DeepSeek - 隐私政策
  • Kimi (月之暗面) - 隐私政策
  • 智谱 AI - 隐私政策
  • OpenAI - 隐私政策

使用这些服务时，您的数据将受到相应服务商隐私政策的约束。

6. 数据安全

  • API Key 使用 iOS Keychain 或 UserDefaults 安全存储
  • 所有网络请求使用 HTTPS 加密传输
  • 应用沙盒机制保护本地数据

7. 儿童隐私

本应用不针对 13 岁以下儿童，也不会故意收集儿童的任何信息。

8. 隐私政策变更

如果我们对本隐私政策进行重大变更，我们会通过应用内通知或更新日志告知您。

9. 您的权利

您可以随时：

  • 在设置中查看和删除您的 API 配置
  • 清空对话历史和会话记录
  • 清空工作区文件
  • 卸载应用以删除所有本地数据

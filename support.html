❓ 常见问题

Q: 为什么显示"API Key 未设置"？
A: 请前往「设置」页面配置您的 LLM 服务商和 API Key。本应用需要您自行提供 API Key 才能使用。
Q: 如何获取 API Key？
A: 访问对应服务商官网注册账号：DeepSeek (deepseek.com)、Kimi (moonshot.cn)、智谱 (zhipuai.cn)，在控制台创建 API Key。
Q: 显示"HTTP 401"错误？
A: API Key 无效或已过期，请检查是否正确复制，或前往服务商控制台重新生成。
Q: 生成速度很慢？
A: 创建 Skill 需要多轮 AI 对话，复杂的 Skill 可能需要 1-3 分钟。您可以在进度条看到实时状态。
Q: 如何导出生成的 Skill？
A: 进入「工作区」标签页，点击右上角导出按钮，即可将所有文件打包为 ZIP。
Q: 数据存储在哪里？
A: 所有数据（包括 API Key、对话历史、生成的文件）都只存储在您的设备本地，不会上传到任何服务器。
Q: 支持 iPad 吗？
A: 目前为 iPhone 优化，iPad 可兼容运行。

💡 使用技巧

  • 使用内置模板可以更快开始，它们已经包含了优化的提示词
  • 描述 Skill 时越具体越好，包括目标用户、功能需求、输出格式等
  • 如果 AI 提出结构化问题，按向导一步步填写可以获得更精准的结果
  • 生成完成后可以在工作区预览和编辑文件
  • 推荐使用 DeepSeek 或 Kimi，性价比高且中文能力强
